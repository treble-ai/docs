**¿Qué hace Trebot?**

Trebot sincroniza conversaciones de tus aplicaciones de WhatsApp (personal o business) con Hubspot, sin necesidad de API o código, diariamente. El historial de cada conversación se guarda en los contactos, junto a un resumen de que se habló en ese día. Trebot también te puede informar sobre tus tareas de Hubspot todas las mañanas, para que empieces tu día con el contexto necesario para sacar adelante todas tus actividades más importantes.

**¿Cómo empezar?**

Sólo necesitas una cuenta de HubSpot (Free, Starter o Professional), WhatsApp instalado en tu teléfono y el navegador Chrome para las funciones de la extensión. No se requiere acceso a API ni software adicional.

La configuración es rápida y sencilla, tomando menos de 3 minutos. Simplemente:

1. Instala Trebot desde [HubSpot Marketplace](https://ecosystem.hubspot.com/marketplace/apps/whatsapp-sync-con-trebot).

   ![Captura de Tela 2025-05-20 às 13.16.03.png](/images/CapturadeTela2025-05-20às13.16.03.png)
2. Escanea el código QR con tu WhatsApp

   ![Capturade Tela2025 05 20às13 17 06 Pn](/images/CapturadeTela2025-05-20às13.17.06.png)
3. Instala la extensión de Chrome (opcional) e invita a tu equipo para que también registren sus conversaciones en Hubspot.

   ![image.png](/images/image.png)
4. Elige tus preferencias de sincronización a partir del mensaje que recibirás en tu WhatsApp y listo.
